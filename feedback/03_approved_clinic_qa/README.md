# 03 Approved Clinic Q&A

Use this folder for doctor-approved human-readable Q&A.

This is not yet the final machine-ready corpus. These files are the clean approved source used to create RAG-ready documents.

Recommended format:

```markdown
## Question
Patient question here.

## Approved Answer
Doctor-approved answer here.

## Tags
ivf, fertility assessment, appointment

## Safety Notes
Mention appointment fallback if personal advice is needed.
```

Filename rule:

```text
YYYY_MM_DD_approved_clinic_qa_short_topic_doctor_approved.md
```

Example:

```text
2026_06_27_approved_clinic_qa_ivf_success_doctor_approved.md
```
