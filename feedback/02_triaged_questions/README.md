# 02 Triaged Questions

Use this folder to classify raw doctor feedback.

Recommended labels:

- `approved`
- `needs_doctor_answer`
- `needs_rewrite`
- `out_of_scope`
- `medical_safety_sensitive`
- `needs_web_search_policy`
- `duplicate`

The output of this folder should move into `../03_approved_clinic_qa/` only after review.

Filename rule:

```text
YYYY_MM_DD_triage_notes_short_topic.md
```

Example:

```text
2026_06_27_triage_notes_initial_doctor_review.md
```
